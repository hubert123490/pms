Use Case: Dodaj opinię
Preconditions:
  • Użytkownik musi być zalogowany w systemie.
Postconditions:
  • Opinia wystawiona
Actors:
  • Wynajmujący (tenant)
Main flow:
  1. System wyświetla polę dodania opinii
  2. Użytkownik wybiera wystawienie opinii dla danego właściciela nieruchomości (w przypadku wynajmującego)
  3. Użytkownik podaje swoją opinię
  4. System wyświetla opcję edycji opinii
  5. System wyświetla podsumowanie opinii po edycji
  6. Użytkownik potwierdza i umieszcza opinię
  7. System akceptuje dodanie opinii przez klienta
Alternate flow:
  • 2 Brak właściciela nieruchomości END FLOW
  • 3 Pole opinii jest puste GOTO 2
  • 6 Użytkownik anulował operację dodawania opinii END FLOW

Use Case: Wynajmij apartament
Preconditions:
  • Użytkownik musi być zalogowany w systemie.
Postconditions:
  • Wynajęto apartament
Actors:
  • Wynajmujący (tenant)
Main flow:
  1. System wyświetla dostępne apartamenty w danej lokalizacji
  2. Użytkownik wybiera dany apartament
  3. Użytkownik wypełnia formularz
  4. System sprawdza poprawność wprowadzonych danych
  5. System nawiązuje sesję z danym serwisem bankowym i oczekuje potwierdzenia płatności
  6. System wyświetla podsumowanie wprowadzonych danych i finalizuję transakcję
Alternate flow:
  • 1 Nie znaleziono żadnych nieruchomości NONE
  • 4 Wprowadzone dane są niepoprawne GOTO 3
  • 5 Użytkownik anulował operację lub zerwał sesję z bankiem (nie dokonał płatności) END FLOW

Use Case: Odstąp od wynajmowania danego mieszkania
Preconditions:
  • Użytkownik musi być zalogowany w systemie. Użytkownik musi wynajmować przynajmniej jeden apartament.
Postconditions:
  • Odstąpiono od wynajmu danego mieszkania.
Actors:
  • Wynajmujący (tenant)
Main flow:
  1. System wyświetla wynajmowane apartamenty przez użytkownika
  2. Użytkownik wybiera apartament
  3. Użytkownik wypełnia formularz
  4. System sprawdza poprawność wprowadzonych danych
  5. System usuwa dany apartament z panelu klienta i finalizuje proces
Alternate flow:
  • 1 Nie znaleziono żadnych nieruchomości END FLOW
  • 4 Wprowadzone dane są niepoprawne GOTO 3
  • 4 Termin odstąpienia od wynajmu minął END FLOW
