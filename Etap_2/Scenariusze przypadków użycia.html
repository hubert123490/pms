Grupa 3ID11A
Nazwa: Wprowadź nieruchomość do systemu
Warunki początkowe:
  • Właściciel nieruchomości musi być zalogowany do systemu.
Warunek Końcowy:
  • Nieruchomość wprowadzona do systemu
Aktor:
  • Właściciel
Główny przebieg:
  1. Właściciel wprowadza nazwe obiektu
  2. Właściciel wprowadza adres obiektu(Nazwa ulicy, numer domu,kod pocztowy,miasto)
  3. Właściciel wprowadza podstawowe dane obiektu (ilość sypialni,ilość miejsc do spania,powierzchnia)
  4. Właściciel wprowadza iformacje dodatkowe o nieruchomości(opcjonalne) np darmowe wi-fi, parking
  5. Właściciel wprowadza zasady obiektu zakaz palenia itp zameldowanie/wymeldownie (od-do)
  6. Właściciel wprowadza zdjęcia nieruchomości
  7. Właściciel wprowadza cene za noc
  8. System wyswietla podsumowanie wprowadzonych danych
  9. Właściciel zatwierdza dane
Przebieg alternatywny:
  • 8 Błędnie wprowadzone dane idź do punktu nr 1

Nazwa: Rejestracja użytkownika
Warunki początkowe:
  • użytkownik nie może być zalgowany
Warunek Końcowy:
  • użytkownik wprowadzony do systemu
Aktor:
  • użytkownik niezalogowany
Główny przebieg:
  1. Użytkownik niezalogowany wybiera typ konta do rejestracji (właścieciel/klient)
  2. Użytkownik niezalogowany wprowadza dane potrzebne do rejestracji
  3. Użytkownik akceptuje regulamin
  4. System weryfikuje poprawność wprowadzonych danych
  5. Użytkownik otrzymuje możliwość zalogowania się na podstawie wprowadzonych wcześniej danych
Przebieg alternatywny:
  • 4 Błędnie wprowadzone dane idź do punktu nr 2

Grupa 3ID11A
Nazwa: Dodaj opinię
Warunki początkowe:
  • Użytkownik musi być zalogowany w systemie.
Warunek końcowy:
  • Opinia wystawiona
Aktor:
  • Wynajmujący (tenant)
Główny przebieg:
  1. System wyświetla polę dodania opinii
  2. Użytkownik wybiera wystawienie opinii dla danego właściciela nieruchomości (w przypadku wynajmującego)
  3. Użytkownik podaje swoją opinię
  4. System wyświetla opcję edycji opinii
  5. System wyświetla podsumowanie opinii po edycji
  6. Użytkownik potwierdza i umieszcza opinię
  7. System akceptuje dodanie opinii przez klienta
Przebieg alternatywny:
  • 2 Brak właściciela nieruchomości END FLOW
  • 3 Pole opinii jest puste GOTO 2
  • 6 Użytkownik anulował operację dodawania opinii END FLOW

Grupa 3ID11A
Nazwa: Wynajmij apartament
Warunki początkowe:
  • Użytkownik musi być zalogowany w systemie.
Warunek końcowy:
  • Wynajęto apartament
Aktor:
  • Wynajmujący (tenant)
Główny przebieg:
  1. System wyświetla dostępne apartamenty w danej lokalizacji
  2. Użytkownik wybiera dany apartament
  3. Użytkownik wypełnia formularz
  4. System sprawdza poprawność wprowadzonych danych
  5. System nawiązuje sesję z danym serwisem bankowym i oczekuje potwierdzenia płatności
  6. System wyświetla podsumowanie wprowadzonych danych i finalizuję transakcję
Przebieg alternatywny:
  • 1 Nie znaleziono żadnych nieruchomości NONE
  • 4 Wprowadzone dane są niepoprawne GOTO 3
  • 5 Użytkownik anulował operację lub zerwał sesję z bankiem (nie dokonał płatności) END FLOW

Grupa 3ID11A
Nazwa: Odstąp od wynajmowania danego mieszkania
Warunki początkowe:
  • Użytkownik musi być zalogowany w systemie. Użytkownik musi wynajmować przynajmniej jeden apartament.
Warunek końcowy:
  • Odstąpiono od wynajmu danego mieszkania.
Aktor:
  • Wynajmujący (tenant)
Główny przebieg:
  1. System wyświetla wynajmowane apartamenty przez użytkownika
  2. Użytkownik wybiera apartament
  3. Użytkownik wypełnia formularz
  4. System sprawdza poprawność wprowadzonych danych
  5. System usuwa dany apartament z panelu klienta i finalizuje proces
Przebieg alternatywny:
  • 1 Nie znaleziono żadnych nieruchomości END FLOW
  • 4 Wprowadzone dane są niepoprawne GOTO 3
  • 4 Termin odstąpienia od wynajmu minął END FLOW
